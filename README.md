# Like_BiliBili
仿哔哩哔哩项目

 SpringBoot <br/>
 Mybatis <br/>
 WebSocket <br/>
 Redis <br/>
 RocketMQ <br/>
 Elasticsearch <br/>

### 文件存储

 用户登录注册 <br/>
 用户发布动态 <br/>
 系统推送订阅动态 <br/>
 关注用户分组 <br/>
 视频断点续传 <br/>
 文件分片在线播放 <br/>
 视频投币、评论、点赞和收藏 <br/>
 发布弹幕、弹幕的推送、弹幕列表查询 <br/>
 弹幕遮罩功能 <br/>
 获取实时在线人数 <br/>
 ES实现全文搜索功能 <br/>
 添加视频播放记录、查看视频播放量 <br/>
 视频内容推荐功能 <br/>
